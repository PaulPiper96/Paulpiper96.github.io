Sorting Algorithms in Unreal Engine

Task:

Presentation about potenial digital innovations for teaching and learning
‍

Idea:

Thinking about a topic i had to learn myself and a way to improve that expierence
‍

Sorting algorithms arrange data for efficient processing. They enable easy identification of patterns and trends. There are many types, including bubble sort, merge sort, and quick sort. The algorithm used depends on the data set size and desired speed. Sorting is essential for many applications, from finance to scientific research. Learning them isn't the hard part, what is hard is remembering the differences between them. So I build a visual representation using UE.

UE 4.27, C++, Quixel, Zbrush, Substance Painter

Blockchain based social network

Task:

Thesis Project for my bachelors degree in mediainformatics

Idea:

public health/social network app written in Kotlin that uses a Blockchain in the backend

In 2020 i read "the surprising power of our social networks and how they shape our lives" by James H. Fowler and Nicholas Christakis. That inspired an attempt to write an application for a social network that dosen't sell user data. I thought that a blockchain would provide a way to give people the power over their data, without comprising the User-Expierence. One aspect of this application was to store deletable data on a blockchain that would be needed in europe because of the "DSGVO". In retrospective i would add, that the "solid" project by Tim Berners-Lee could provide a big benefit, that i wasn't aware of at that time.

Android Studio, Bonebits-Blockchain, Kotlin, Adobe XD
